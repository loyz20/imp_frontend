# Struktur Folder Standar Industri

- src/components: Komponen UI reusable
- src/pages: Halaman aplikasi
- src/layouts: Layout aplikasi
- src/hooks: Custom React hooks
- src/utils: Utility functions
- src/services: Service layer (API, logic)
- src/store: State management (Redux, Zustand, dsb)
- src/constants: Konstanta global
- src/types: TypeScript types (jika pakai TS)
- src/styles: Global/local styles
- src/assets: Asset statis (gambar, font, dsb)
- src/api: API layer (request, endpoint)
- src/routes: Routing aplikasi
- src/middlewares: Middleware (jika perlu)
- src/tests: Unit/integration tests
- src/config: Konfigurasi environment
